## How to Contribute

We welcome contributions from the community! Here's how you can help:

### Ways to Contribute

1. **Report Bugs** - Found a bug? [Create an issue](https://github.com/koriandyr/bulk-rename/issues/new)
2. **Suggest Features** - Have an idea? Share it in our [issues](https://github.com/koriandyr/bulk-rename/issues)
3. **Submit Pull Requests** - Fix bugs or add features (see [CONTRIBUTING.md](CONTRIBUTING.md))
4. **Improve Documentation** - Help make our docs better
5. **Share Feedback** - Let us know how you're using bulk-rename

### Recognition

Contributors will be added to this list once their pull requests are merged. We recognize all forms of contribution including:

- Code contributions
- Documentation improvements
- Bug reports with detailed reproduction steps
- Feature suggestions and design discussions
- Helping others in issues and discussions
