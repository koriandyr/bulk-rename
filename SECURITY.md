# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| >= 1.0  | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

Vulnerabilities can be reported through Github Discussions currently.

The bulk-rename tool is a simple piece of automation shared as-is. Vulnerability reports will be reviewed and approrpriate fixes entertained when required.
